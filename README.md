## Run Locally

Clone the project

```bash
  git clone https://github.com/cloud179200/seller.git
```

Go to the project directory

```bash
  cd seller
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

build

```bash
  npm run build
```
